Blockwaves

Blockwaves is a concept by koder for Minecraft 1.11.
Blockwaves allows you to create a shockwave when you hit the ground, flinging nearby blocks into the air.
Blockwaves can only affect one block type at a time, but the command can be configured so you can choose which block type it affects.

Installation

Copy the command, then paste it into a command block in your 1.11 Minecraft world.
Activate the command block by setting it to Always Active or by powering it from the side with redstone.
The module installs, and now you can create your very own blockwaves. Enjoy!
