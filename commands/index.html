Data Packs

Quickfire Bows
This data pack speeds up arrows to the maximum allowed speed once fired, regardless of how far you pull the bow back.

Rotation Calculator
This data pack calculates an entity's horizontal and vertical rotation and saves them to scoreboard values.


One-Commands

Blockwaves
This command block concept creates shockwaves when you land on a block, launching nearby blocks into the air.

Falling Logs
This contraption causes log blocks to drop down like sand when a log beneath them is chopped, instead of floating in the air.

Halfcraft
Halfcraft allows you to free any block from the grid, then move the blocks around the world pixel by pixel.

Sign Command Toggle
This command creates a sign-activated toggle for command block chains, which can be easily installed and added on to your chain.
