Sign Command Toggle

This is a simple command tool by koder for Minecraft 1.9.
With this tool, you can create a sign-based switch for a line of commands. It has a few advantages over the switch, however.
One advantage is that the signs do not affect tick order at all, so they can be turned off and on without messing up the order of your commands.
Another advantage is that only ops can use this sign toggle. Finally, the player can also use the sign to describe their line of commands.

Installation

Copy the command, then paste it into a command block negative 1 block in the z axis from a sign.
See the video above for specific instructions and a visual example of where exactly to put the sign and the command block.
Activate the command block by setting it to Always Active or by powering it from the side with redstone, then the sign will become a 'switch' affecting the line of commands behind it.
