Falling Logs

Falling Logs is a command contraption by koder for Minecraft 1.9.
The way it works is simple; You chop down a log block and the logs above it will fall down, allowing for easier and more interesting tree-chopping. Enjoy!

Installation

Copy the command below, then paste it into a command block in your 1.9 Minecraft world.
Activate the command block by setting it to Always Active or by powering it from the side with redstone, and the commands will spawn.
To uninstall, simply click the [Uninstall] sign.
