Rotation Calculator

The Rotation calculator is a system built by koder for Minecraft 1.12.
Once set up in your world, it can be used to save any entity's horizontal and vertical rotation to scoreboard values with a simple function call.
All commands, videos and resources related to the Rotation Calculator can be found on this page.

Installation & Usage

Download
Download the .zip file using the button above and extract it. Place the koder folder into your Minecraft world's data\functions folder. A quick way of finding your world folder is to select that world from Minecraft's world select menu, then click Edit and click Open World Folder. If your world doesn't have a data or functions folder, just create them.
Load up the world in Minecraft 1.12, and type /function koder:rot_calc/install into the chat. This function creates the 2 scoreboards that the horizontal and vertical rotations of entities will be saved to.
To calculate an entity's rotation, simply execute from that entity to run the koder:rot_calc/tick function, like this:
/execute @e[entities you want to calculate] ~ ~ ~ function koder:rot_calc/tick
The entities' horizontal rotations will be stored to the krc-rh scoreboard, and the vertical rotations stored to the krc-rv scoreboard.
