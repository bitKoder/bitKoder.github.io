Resource Packs

Below is a list of all the resource packs I have released to date. You can find the full video and a short description for each resource pack below the title. To the right of each title is a download button for that resource pack.

Basic Command Block Textures

These are the Command Block textures that I use in all of my videos since about the start of 2016. They're a very simplified and clean version of the command blocks in the default texture pack, and use the default 16x16 resolution. They are also perfectly symmetrical, and flash white instead of the default multicolor flashing.

Switches

This resource pack changes the lever model to a switch model. In the center of the switch is a red/green indicator, showing whether the switch is on or off. The switch's appearance fits more with the stone button, but kinda doesn't work with its crafting recipe.

Clean UI

This pack changes the look of the UI (or User Interface) to give it a cleaner, more simplistic look.

3D Items

A simple resource pack that changes item's model files to make them display in 3D in the inventory. Certain items extend outside their cell boundaries because of their height. This resource pack changes no textures and is compatible with all other resource packs.
